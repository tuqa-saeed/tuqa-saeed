### KPI Dashboard — UiPath Orchestrator API

![UiPath](https://img.shields.io/badge/UiPath-Orchestrator_API-%23FA4616?style=flat-square&logo=uipath&logoColor=white)
![Slack](https://img.shields.io/badge/Slack-Alerts_Integration-%234A154B?style=flat-square&logo=slack&logoColor=white)
![Dashboard](https://img.shields.io/badge/Type-KPI_Dashboard-%236D28D9?style=flat-square)

Built a KPI dashboard that monitors robot activity using the **UiPath Orchestrator API**. The dashboard tracks error and fatal messages in real time and sends automated alerts to **Slack** whenever a robot stops or encounters issues — helping the team detect failures faster and improve automation monitoring.
